# FiveLeagues
```
Procedurally generated adventure game 

Expected Behaviors from MAIN MENU:
New Game
  1) Ask if generating a party or loading one\
  1a) If generating a party, use number of casters provided by USER (1::4)
      Generated Party will consist of 4 heros and 2 followers
  2) Allow USER to allocate initial equipment to heros and followers
  3) Generate Campaign
  4) Follow Five Leagues Beyond the Borderlands phases of gameplay
  4a) Preparation Stage
     1) Save Campaign STATE
     2) Start Play from last STATE
     3) Local Events
     4) Hard Times
     5) Campaign Activites
     6) Trade
     7) Research
     8) Decide Adventure
     9) Outfit for Adventure
  4b) Adventures Stage
    1) Encounter Locations
    2) Unexplored Locations
  4c) Encounters Stage
    1) Traveler Encounters
    2) Battle Encounters
    3) Scenario Types
    4) Ending Scenario
  4d) Resolution Stage
    1) Experience
    2) Loot
    3) Enemy Plans
    4) Victory and Defeat
    5) The Next Campaign
```
